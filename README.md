A fun horse racing game built with **Vue 3, TypeScript, and Vuex 4**.

## Live Demo

Check out the live game here: [https://vocal-crumble-3a3534.netlify.app/](https://vocal-crumble-3a3534.netlify.app/)

## Features

- Generate random horses with unique names and colors
- Create race schedules dynamically
- Simulate races with animated horse positions
- View results and track rounds
- Modal notifications for program/race state

## Tech Stack

- Vue 3 + Composition API
- TypeScript
- Vuex 4 (state management)
- Vite (build tool)
- Vitest (unit tests)
- Netlify (deployment)

## Type Support for `.vue` Imports in TS

TypeScript cannot handle type information for `.vue` imports by default, so we replace the `tsc` CLI with `vue-tsc` for type checking. In editors, we need [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) to make the TypeScript language service aware of `.vue` types.
